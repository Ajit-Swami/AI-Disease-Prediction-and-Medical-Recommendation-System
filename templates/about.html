About the Project

The Disease Prediction and Medicine Recommendation System is a smart web-based platform built using Python, Flask, and MySQL. It uses Artificial Intelligence (AI) solely to calculate the confidence level of the predicted disease based on user symptoms. Once the prediction is made, the system fetches relevant medications, precautions, diet, and workouts directly from the database. Additional features include PDF download, voice input, and secure login/logout functionality.

Technologies Used

  • Python & Flask (Backend) & AI
  • MySQL (Database)
  • CSS, Bootstrap (Styling)
  • Speech Recognition (Voice Input)
  • PDF Export (html2pdf.js)
